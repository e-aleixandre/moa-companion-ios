# Moa Companion for iOS

Private iPhone/CarPlay companion for Moa Ops Assistant.

The app is a thin, authenticated client of Moa's server-owned Ops layer. It will provide verified operational briefings, safe directed instructions, and later push-to-talk. It does not execute agents, approve permissions, or make production decisions locally.

## Status

Early architecture/protocol scaffold. iOS builds and device validation will run in macOS CI/Xcode; no server-to-Mac remote access is required.
